# Control & Predict

**An interactive educational app designed to help children with autism develop predictive thinking and emotional regulation skills through engaging animations and activities.**

---

## What Is This App?

Control & Predict is a therapeutic web application that helps children with autism spectrum disorder (ASD) understand cause-and-effect relationships in everyday situations. Through interactive animated scenarios, children learn to predict what happens next, building crucial cognitive and social skills.

The app provides a safe, visually engaging environment where children can:
- **Watch** familiar scenarios unfold
- **Predict** what happens next by choosing from options
- **Learn** from immediate, gentle feedback
- **Practice** with repetition in a stress-free setting

---

##  Why Is It Useful?

### For Children with Autism

| Challenge | How This App Helps |
|-----------|-------------------|
| **Difficulty predicting outcomes** | Visual scenarios with pause points let children practice predicting what comes next |
| **Anxiety in new situations** | Familiar locations (park, home, school) build confidence through repetition |
| **Emotional regulation** | Built-in calming corner with sensory activities |
| **Daily routine struggles** | Visual routine builder with timers and practice mode |
| **Social story comprehension** | Interactive social stories for common situations |

### Research-Based Approach

This app is built on evidence-based practices for autism education:
- **Visual learning** - Uses animations instead of text-heavy instructions
- **Predictability** - Consistent structure reduces anxiety
- **Immediate feedback** - Clear visual cues for correct/incorrect responses
- **Positive reinforcement** - Reward system with collectible stickers
- **Sensory considerations** - Calming activities for self-regulation
